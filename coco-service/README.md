# COCO Service - IT咨询服务网站

COCO Service是一个提供专业IT咨询、系统架构优化和技术效率提升服务的网站。

## 功能特点

- 响应式设计，适配各种设备
- 详细的服务介绍和价格信息
- 集成Stripe支付系统，支持在线支付
- 完整的咨询服务预约流程

## 技术栈

- Next.js 13+ (App Router)
- React 18
- Tailwind CSS
- Stripe 支付API
- Vercel部署
